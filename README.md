Esse repositorio é para adicionar um relógio em iframe no Obsidian, feito em html.
Faça uma bifurcação do repositorio e customize como quiser.

## Como usar o iframe no Obsidian

Copie o endereço do arquivo e cole no seu arquivo desejado no Obsidian.

```
<iframe src="https://joaopedro-chaves.github.io/clock-iframe-obisidian/" width="100%" height="100px"></iframe>
```

## Git Clone

```
git clone https://github.com/joaopedro-chaves/clock-iframe-obisidian
```

## Deploy

Use o Github Pages para hospedar o arquivo direto no repositorio, ou se preferir use o Vercel, etc...
